Four papers this week that cut inference costs without cutting quality.

⚡ FlashKV (2604.11432
🔗 https://arxiv.org/abs/2604.11432)
Dynamic KV-cache eviction using attention entropy — drops memory 40% on long-context models with minimal perplexity hit. MIT license, vLLM integration ready.
Watch: Now — if you serve 100K+ context windows, this is immediate ROI.

🎯 SpecDraft-Distill (2604.10987
🔗 https://arxiv.org/abs/2604.10987)
Distills tiny draft models (10x smaller) specifically for speculative decoding. 2.3x speedup on LLaMA-3-70B without changing the target model.
Watch: Now — drop-in for existing inference stacks.

📐 MatFormer (2604.11245
🔗 https://arxiv.org/abs/2604.11245)
Specialized architecture for math-in-code reasoning. Strong results on MATH-Code (15% lift) but currently Python-only and unstable on edge cases.
Watch: 6 months — wait for the v2 implementation.

🧮 TopoNN (2604.10876
🔗 https://arxiv.org/abs/2604.10876)
Persistent homology meets graph neural networks. Theoretically elegant for capturing multi-scale structure, but empirical validation is thin.
Watch: Academic — track if you do geometric deep learning.
